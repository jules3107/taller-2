Definiciones Backend

Backend

Parte de una aplicación web que maneja lógica del servidor y la base de datos.

API

Interfaz que permite la comunicación entre diferentes sistemas

Base de datos

Sistemas donde se almacenan y gestionan los datos de la aplicación


Framework Backend

Conjunto de herramientas y librerías para desarrollar servidores y aplicaciones web.

Servidor

Computadora o software que maneja peticiones y entrega de datos a los clientes.
